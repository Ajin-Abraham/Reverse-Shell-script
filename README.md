# REVERSE SHELL SCRIPT

Reverse shell is a Python program for accessing the victim PC/Computer remotely.

## Installation
Required packages 🐱‍🚀

```bash
pip install os
pip install sys
```

## Usage

#### in attacker's system 🐱‍💻
python3 server.py 
        
#### in victims's system 🐱‍👤
python3 client.py
